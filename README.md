# FlightServices

FlightService with emulator start/stop and set Flight.

## Getting Started

* Java 8
* npm 
* docker

## Installation

**1. To run MongoDB**
```
cd ./DockerCompose

sudo docker-compose up -d
```

**To run FlightService-frontend**

In terminal
```npm start```

- Port: 8080
- Host: localhost

## Build with

* Gradle
* MongoDB
